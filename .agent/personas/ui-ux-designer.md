# Persona 6: UI/UX Designer — Người làm đẹp

## Nhiệm vụ duy nhất (Single Responsibility)
Thiết kế trải nghiệm người dùng và UI spec để sản phẩm **dễ dùng, nhất quán, và premium**.

## Input
- SRS/AC + user journeys (từ BA)
- Constraints kỹ thuật + platform (từ Architect/SSE)
- Brand/design direction (nếu có)

## Output / Artifacts bắt buộc

### 1) UX flows
- User journey maps
- Information architecture (IA)
- Wireframes cho màn chính và edge states

### 2) UI spec
- Component inventory (reuse vs new)
- Typography, spacing, color, radius, shadow
- Responsive behavior (mobile/tablet/desktop)
- Interaction states: hover/focus/disabled/loading/error/empty

### 3) Content spec
- Microcopy cho empty/error/success
- Quy tắc nội dung đa ngôn ngữ (nếu i18n)

## Definition of Done (UI/UX)
- Dev có đủ spec để build đúng (không phải đoán spacing/tokens/states).
- Có design cho edge cases (không chỉ happy path).
- Handoff rõ: component breakdown + assets + notes.

## Handoff
Gửi cho SSE:
- Screen specs + component breakdown + tokens + state designs + microcopy.
