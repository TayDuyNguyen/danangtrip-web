# Decisions log

Mỗi quyết định kỹ thuật “định hướng” nên có 1 file riêng để truy vết.

## Template
- **Date**: YYYY-MM-DD
- **Decision**: mô tả ngắn
- **Context**: tại sao cần quyết định
- **Options**: 2-3 phương án + trade-offs
- **Chosen**: phương án chọn + lý do
- **Consequences**: ảnh hưởng lên codebase/ops

## Naming
`YYYY-MM-DD__short-slug.md`
